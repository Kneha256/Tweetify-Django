# Tweetify-Django

Tweetify is a simple Django-based social media application where users can:

- Register and log in  
- Create tweets with text and images  
- Edit and delete their own tweets  
- View tweets in a responsive Bootstrap grid  

## Tech Stack
- Python  
- Django  
- Bootstrap  
- SQLite
